CSS unit & Measurements

CSS provide various units to define dimensions, spacing ad typograpy.These units are categorized into absolute and relative types.

1. Absolute Units

Absolute units define fixed sizes that do not change relative to the viewport or parent elements.

  • px - pixels
  • cm - Centimeters
  • mm - millimeters
  • in - inches

2. Relative units --- Responsive design

Relative units depend on parent elements, viewport size, or root elements, making the ideal for responsive design.

em and rem are font relative units. em unit is relative to the parent element font size and rem is relative to root element (html) font size.

Percentage (%) units are relative to the parent element(width and height)

vw (viewport width) and vh(viewport height) units are to the viewport/available screensize
