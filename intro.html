css - Cascading Sytle Sheet

CSS is a programming language used to improve the presentation of a html page

Css will help modify aspects like texts sizes,colors,backgrounds,image sizes and shapes, boarders,positions,responsiveness(different looks on different screen sizes) among other aspects of presentation

CSS is used to desinged user interfaces for web pages and it helps enhance user experience.

CSS Syntax(how do we write css)

A selector is used to target an element ora group of elements for styling

{} - A declaration block - groups proroperties and values to be applied on the selector

Value - the final look you are interested in for the page (red, fullpage,400px...) 

    
        selector{
            property: value;
            property: value;
        }
    
    
Example

    
        h1{
            color: green;
        }
