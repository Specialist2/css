Colors

CSS supports multiple ways to define colors

  • Named colors e.g green.,blue,...
  • Hex code (Hexadecimal)
  • RGB - (red green blue) rgb(23,0,255)-- allowed values (0-255)
  • RGBA - a (alpha - transparency level/opacity) - rgba(255,255,255, 0.7)
  • HSL (Hue, Satureation, Lightness) hsl(240, 100%,50%)-Blue
